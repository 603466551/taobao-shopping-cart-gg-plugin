# 淘宝购物车导出助手

这是一个本地 Chrome 扩展，用来在已登录的淘宝/天猫购物车页面中识别商品信息，并导出为 Excel 可打开的 `.xls` 表格。

## 安装

1. 打开 Chrome，进入 `chrome://extensions/`。
2. 开启右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录：`C:\Users\60346\Documents\taobao`。

## 使用

1. 打开淘宝购物车页面并确认商品已经加载完成。
2. 点击浏览器右上角的“淘宝购物车导出助手”图标。
3. 点击“扫描预览”。
4. 在弹窗预览列表中确认商品、图片、价格，并手动勾选要导出的商品。
5. 点击“导出选中”。
6. 浏览器会下载一个 `淘宝购物车_时间.xls` 文件。

## 导出字段

- 店铺
- 商品名称
- 规格
- 单价
- 数量
- 小计
- 商品链接
- 图片链接
- 是否选中
- 抓取时间

## 说明

淘宝页面结构会不定期调整，插件会优先从购物车渲染区域内识别商品容器，并过滤推荐、广告、热卖等区域，避免把购物车下方内容导出进去。

弹窗关闭后，再次打开会尝试从当前浏览器会话中恢复上一次扫描结果和勾选状态。

图片会优先从淘宝购物车商品图结构中提取，例如 `imageContainer` 内的 `img.image`。重复候选会按商品 ID 和 SKU/购物车参数合并，避免把不同规格误合并，也避免把活动图、横幅图错配成商品图。
